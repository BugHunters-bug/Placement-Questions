Question: 

Given an array arr [] of size N. The task is to find the first repeating element in the array of integers,
i.e, an element occurs that occurs more than once whose index of the first occurence is smallest.

Constraints:
1<=N<=1e6
0<=Ai<=1e6

Example:

input:
7
1 5 3 4 3 5 6

Output:
2

Explanation:
5 is appearing twice and its first appearance is at index 2 which is less than 3 whose first occuring index
is 3.
